Projeto portfólio modelo Anna Bella

Foi contruído, através dos conhecimentos obtidos até o momento do curso, um website portfólio para uma modedo fictícia chamada Anna Bella.

As fotos foram cedidas pelo curso.

O website contém 4 páginas, sendo elas: home, biografia, companhas publicitárias e contato.

Curso Udemy: Domine Web - 20 Cursos - HTML5, CSS3, SASS, Bootstrap, JS, ES6, PHP 7, OO, MySQL, JQuery, MVC, APIs, IONIC e muito mais.
